# Créer un site

1. Depuis le tableau de bord, clique sur **« Créer un site »**.
2. Choisis un **template** dans la galerie (le bouton « Continuer » reste désactivé tant qu'aucun template n'est sélectionné).
3. Donne un **nom** à ton site.
4. Clique sur **« Créer »** pour générer le site.

## Pourquoi le bouton « Continuer » est désactivé

Le bouton **Continuer** de l'étape template est grisé **tant qu'aucun template n'est sélectionné**. Sélectionne une carte de template pour l'activer.

## Pourquoi le bouton « Créer » est désactivé

Le bouton **Créer** est désactivé **si le champ nom est vide ou invalide** (le nom ne peut contenir que lettres, chiffres et tirets).
