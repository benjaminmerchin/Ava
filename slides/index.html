How we built it

Production-grade, by design.

What we used
  • 3-agent crew: Perception ∥ Knowledge → Guide (+ a lean 1-agent fast path)
  • Tool calling — a doc_search RAG tool over the product docs
  • Structured output (output_pydantic): {speech · highlight · next_step}
  • Parallel tasks — Perception & Knowledge run at once
What we used
  • AI Gateway — every LLM call routes through it (OpenAI-compatible)
  • Multi-model — Claude Opus 4.8 / Sonnet / Haiku behind one endpoint
  • Resilient routing — Bedrock Anthropic was gated → we switched to direct Anthropic in one line
  • Observability — every agent call traced in the dashboard
DOM state → crew → { speech · highlight · next step } reliability eval: 12/12 ✓
The solution

Ava — support that sees your screen.
